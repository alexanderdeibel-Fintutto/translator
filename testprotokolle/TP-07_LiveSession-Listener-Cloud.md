## Testaufgaben

### A. Session beitreten

| Nr. | Aufgabe | Status | Bemerkung |
|-----|---------|--------|-----------|
| A1 | Live → **Listener** wählen | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| A2 | **Session-Code** manuell eingeben → Beitritt funktioniert | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| A3 | Alternativ: **QR-Code scannen** → automatischer Beitritt | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| A4 | **Zielsprache** wählen (z.B. Englisch) | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |

### B. Übersetzungen empfangen

| Nr. | Aufgabe | Status | Bemerkung |
|-----|---------|--------|-----------|
| B1 | Speaker spricht → **Übersetzung erscheint** auf dem Listener-Display | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| B2 | Übersetzung ist **groß und gut lesbar** angezeigt | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| B3 | Neue Übersetzungen ersetzen die vorherige (aktuelle immer oben/groß) | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| B4 | **Verlauf** der letzten Übersetzungen ist sichtbar (scrollbar) | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| B5 | "Warte auf Übersetzung..." wird angezeigt, wenn Speaker still ist | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |

### C. Auto-TTS (automatisches Vorlesen)

| Nr. | Aufgabe | Status | Bemerkung |
|-----|---------|--------|-----------|
| C1 | **Auto-TTS Toggle** finden und prüfen (Standard: an?) | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| C2 | Speaker spricht → Listener hört die **Übersetzung automatisch vorgelesen** | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| C3 | Auto-TTS **deaktivieren** → Übersetzungen werden NICHT mehr vorgelesen | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| C4 | Auto-TTS wieder **aktivieren** → nächste Übersetzung wird wieder vorgelesen | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
