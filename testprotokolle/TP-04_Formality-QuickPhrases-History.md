| A2 | Zielsprache **Deutsch**, Toggle auf "Sie" → Text EN→DE: "How are you?" → enthält "Sie" / "Ihnen" | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
| A3 | Toggle auf "Du" → gleicher Text → enthält jetzt "du" / "dir" | ☐ OK  ☐ FEHLER  ☐ TEILWEISE | |
