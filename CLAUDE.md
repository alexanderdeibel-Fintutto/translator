# Project Guidelines

## Communication
- WICHTIG: Keine deutschen Sonderzeichen verwenden (kein ae/oe/ue als Umlaute, kein sharp-s). Der User kann diese Zeichen nicht sehen/eingeben. Immer Ersatzschreibweise nutzen: ae statt ä, oe statt ö, ue statt ü, ss statt ß.
- Antworten auf Deutsch, wenn der User Deutsch schreibt.
